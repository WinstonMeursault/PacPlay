# PacPlay

局域网内游戏平台 🎮

## 快速开始

### 手动构建

#### 下载并构建项目

克隆仓库

```bash
git clone https://github.com/WinstonMeursault/PacPlay.git ./PacPlay
cd ./PacPlay
```

随后执行 ```make all``` 构建项目

#### 部署

在一台主机上执行 ```make run-server``` 运行服务器端

在另外的主机上执行 ```make run-client``` 即可运行客户端
